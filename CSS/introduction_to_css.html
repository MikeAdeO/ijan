TYPES OF CSS SELECTOR

  • CSS ELEMENT SELECTOR
  • CSS ID SELECTOR
  • CSS CLASS SELECTOR
  • CSS UNIVERSAL SELECTOR
  • CSS GROUP SELECTOR

CSS ELEMENT SELECTOR

Element selector selects the HTML element by name

CSS ID SELECTOR

The id selector selects the Id attribute of an HTML element to select a specific element. The ID is represented in the style with the #

CSS CLASS SELECTOR

The class selector selects HTML element with a specific class attribute.awayIt is used with a period character or a fullstop or a dot

CSS UNIVERSAL SELECTOR

The universal selector is used as a wildcard character. It selects ALL element on the website

CSS GROUP SELECTOR

The grouping selector is used to select all the elements with the same style definitions.

A group selector is used to minimize the code . commas are used to seperate each selector in the group
