CSS ANIMATION

CSS ANimation is used to create animation on the webpage.

How do you create Animation

the animation is created in ther @keyframe rule. It is used to control the intermediate steps in a css animation sequence

How animation works

the animation is created in the @keyframe rule , it must be bound with selector , otherwise , it will have no effect.

THE CSS ANIMATION PROPERTIES

  • @keyframes
    • It is used to specify the animation
  • animation
    • This is a shorthand property used for setting all properties , except the animation-play-state and the animation-fill-mode property
  • animation-delay : it specifies when the animation starts
  • animation-direction : it specifies if or not the an imation should play in reserve or alternate cysle
  • animation-duration : it specifies the time duration taken by the taken by the animation to complete
  • animation-fill-mode : it specifies the static style of the element ( when thye animation is not playinmg)
  • animation-iteration-count : it speifies the number of times the animation should be played
  • animation-play-state: it specifies if the animation is running or paused
  • animation-name : it specifies the name of @keyframes animation
  • animation-timing-function: it specifies the speed curve of the animation

@keyframes Rule

the css keyframes rules specifies the animation rule that define the CSS properties at each state with a timeline

@keyframes animation-name { keyframes-selector { css-styles } }

keyframes timing function

this is used to control the animattion rate, we can use it the following vales

  • linear: It means the transition rate will be constant from start to end
  • ease: it means, the animation starts sloly and the after a time period , the speed increases and before it ends , the speed will be slow again
