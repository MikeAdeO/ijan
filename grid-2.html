  • grid-template-columns: It is used to specify the sizes of the column
  • grid-template-rows: it is used to specify the row size
  • grid-template-areas: it is used to sepecify the grid layout by using the named items
  • grid-auto-rows : it is used to specify the the automatic size of rows
  • grid-auto-columns: it is used to specify the automatic size of columns
  • grid-auto-flow: it is used to specify how to place auto-placed items and automatic row size
